# weather information website

→ Brief Description
  this website is a weather information website that fetches data from the OpenWeatherMap API from https://openweathermap.org/current. The user interface includes a search bar for city input, with weather details displayed in a visually appealing format, featuring city name, temperature, weather description, humidity, and wind speed. Overall, the app offers a user-friendly experience for obtaining and visualizing weather information.

→ Framework, tools, and libraries used
  1. React.js
  2. Axios
  3. Framer Motion
  4. CSS
  5. ReactDOM:
  6. OpenWeatherMap API

→ Additional notes or remarks you would like us to consider.
  To enjoy the website, you enter a city name from all around the world and the website will be giving you the weather information. The API used here is the Current Weather Data API from https://openweathermap.org/current.
